xr-aus

Goals

  • New developers that understand HTML, utility CSS and vanilla JavaScript can be productive in a short amount of time.
  • Website content producers can create flexible layouts with minimal training, no HTML knowledge required.
  • The website is built in a way that allows all content and supporting data to be editable via a CMS.
  • The website and CMS have a minimal environmental impact.
  • Website a11y is not an afterthought.
  • Any 3rd parties used (analytics, etc) are privacy-focussed.

Stack

  • The static website is generated with Eleventy.
  • The CSS framework used is Tailwind.
  • The JavaScript framework used is AlpineJS.
  • The live website is deployed to CloudFlare Pages on push to main, and preview deploys are spun up on push to other branches.
  • Content and JSON data updates can be made via the Netlify CMS.
  • OAuth for the Netlify CMS is deployed to CloudFlare Workers, via GitHub Actions.

Local development

git clone https://github.com/d3v1an7/xr-aus
npm install
npm run start

Traps
